FLEXBOX AXES

Le "cross axis" est une ligne imaginaire perpendiculaire à l'axe de la "main"( l'axe originale). Si on met la valeur "row" ou "row-reverse" à la "flex-direction" le "cross axis" est vertical et commence vers le haut, mais si c'est la valeur " column" ou "column-reverse" le "cross axis" est horizontal et commence vers la gauche. On le trouve le plus souvent quand on met à la classe item un height ou un weight.

JUSTIFY-CONTENT

justify-content est une propriété qui permet de justifier les "flex items". Permet aussi de changer comment les éléments sont groupès et spacès. Sa valeur originale est "flex-start" qui signifie que tous les elements sont groupès ensemble au commencement de la cage (container) soit de gauche à droite(row) ou de haut en bas(column). Il y'a aussi la valeur "flex-end" qui groupe tous les elements(items) à la fin de la cage (container) soit de droite à gauche ou de bas en haut. La valeur " center" permet de centrer les elements. Pour espacer chaque element dans la cage on utilise " space-around". Il y'a aussi la valeur " space-between"

ALIGN-ITEMS

pour aligner les éléments "items" tout au long du "cross axis" qui est differente de l'axe original on utilise align-items. sa valeur originale est "stretch". Il y'a aussi la valeur "flex-start", " flex-end", "center".

Overflow des elements horizontale dans une cage "container" on utilise flex-wrap. Sa valeur originale est "nowrap" . Il y'a aussi la valeur "wrap", "wrap-reverse". Il y'a la propriete align-content , qui a "stretch" comme valeur originale. On trouve aussi les values "flex-start", "flex-end", "center", "space-around", "space-between". La propriete flex-basis pour changer le size des flex items. La valeur originale est "auto" avec unite px et %. La propriete flex-grow, sa valeur originale est egale à 0. Il y'a aussi la valeur 1, 2, etc... C'est un element de item. Pour retrecir les elements "items" dans la cage "container" on utilise la propriete flex-shrink, et sa valeur originale est 1. Pour combiner flex-grow, flex-shrink et flex-basis on utilise seulement la propriete flex avec les valeurs comme: "flex: 0 1 auto", c'est une propriete d'item .

CREATING A CSS GRID

- CSS grid est un système de mise en page qui nous permet d'ajouter des éléments à la fois horizontalement et verticalement sur une page Web
- Pour utiliser CSS GRID, on doit créer en premier un parent élément avec la classe grid-container, es: div class = "grid-container".
- Pour les éléments on utilise la classe " grid-item".
- Pour le coder on utilise en CSS la propriete display avec la valeur grid dans la classe conteneur ( container).

CREATING GRID COLUMNS:

- Pour les colonnes on a la propriete grid-template-columns avec l'unitè en px qui est l'unite fixe.
- Les unités du Grid-template-columns : 50px 50px 120px ou auto, il y'a aussi son unité spécial fr(fractional unit), on peut les coder tous ensemble.
- Les éléments de la grille sont affichés en fonction du nombre de colonnes. s'il y a plus d'éléments que de colonnes, les éléments supplémentaires seront mis vers la ligne suivante. En mettant la valeur auto à la propriété grid-template-columns les èléments de la grille seront espacés.
- La valeur fr : fractional unit répartit l'espace disponible du conteneur (container) en fractions. Si on fait la combination de fr avec d'autres unité comme px l'espace laissé aprés l'èlément qui à l'unité px est la meme chose distribué en fractions.
- La valeur de la grille repeat () est un raccourci pour définir des colonnes rapidement et c'est une fonction . Ex: repeat(3, 1fr) -> 3 columns avec chacun 1fr.

CREATING GRID ROWS:

-ON utilise la propriété grid-template-rows pour coder les grilles en ligne (contraire de colonne). Son unité fixe est pixels(px) si on a moins d'éléments de grille (grid items) que les lignes, nous aurons des lignes vides. On a aussi l'unité %, on peut aussi utilisé( en combinant ) fr et px, il existe la valeur auto, la fonction repeat aussi.
